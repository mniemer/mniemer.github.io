  • NU Robotics Lacrosse Goalie

    The goal of this project is to create a robotic lacrosse goalie which will be used by the club lacrosse team at Northwestern.

    I began my role as lead programmer for this project in the fall of 2016. I'm working with a small team to implement a lightning fast vision system which will be integrated with a motor controller. Project files can be found here.

  • Tetris

    I have a deep love of Tetris. In an effort to gain some experience with starting a project from scratch, I decided to build a working local version of my favorite game. I used the pygame package in Python, which proved to be very conenient for managing and drawing sprites.

    I began this project over the summer in 2016, and I have not yet finished. Project files can be found here.

  • Exercising the Numbers

    I worked in a team of 4 to implement various machine learning algorithms in order to accurately predict gym congestion at Northwestern. After compiling a large data set of gym attendance for over a year, we ran multiple classifiers which attempted to use attributes such as weather, date, and time in order to predict the true gym attendance. Our full report can be found here.
